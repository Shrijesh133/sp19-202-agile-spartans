   https://www.getdigital.de/the-code-is-documentation-enough.html

...ist zwar eigentlich mein Motto, trotzdem habe ich den Code
ausführlich kommentiert. Konzeptideen/Lösungen finden Sie in dem
Konzepte_Dokumentation_Aufgaben-Ordner.
Gearbeitet haben wir fast ausschließlich in den Informatikstunden,
wenn die Aufgaben schon alle erledigt waren. Nur in den letzten Tagen
vor der Abgabe habe ich mir das Notebook meiner Mutter geschnappt
(auf meinem PC läuft Greenfoot nicht, da ist Java falsch konfiguriert)
und Zuhause gearbeitet.
Die Grafiken wurden von uns mit den Farben des Originalspiels erstellt,
einige .xcf(Gimp)-Dateien sind auch noch im images-Ordner vorhanden.
(In dieser Ausführung nicht dabei, da der Ordner sonst zu groß wäre)
Der Code ist zu 100% Marke Eigenbau. Bei komplexeren Angelegenheiten,
wie z.B. das Lesen/Schreiben von .txt-Dateien oder Aufrufen von Links
war das Greenfoot-Forum sehr hilfreich.
